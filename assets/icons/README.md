# Icons Directory

Place UI icons and small graphics here.

## Format

**SVG only** - scalable, small file size, can be styled with CSS

## Usage

Icons can be used inline in HTML or as CSS background images:

```html
<!-- Inline SVG -->
<svg class="icon" aria-hidden="true">...</svg>

<!-- IMG tag -->
<img src="assets/icons/search.svg" alt="" role="presentation">
```

## Accessibility

- Decorative icons: Use `aria-hidden="true"` or `role="presentation"`
- Functional icons: Include descriptive text (visible or sr-only)

## Icon Sets

Consider using a minimal icon set or creating custom icons to match the design aesthetic.
