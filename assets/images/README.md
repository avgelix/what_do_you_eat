# Images Directory

Place photos, illustrations, and graphics here.

## Format Guidelines

- **SVG**: Preferred for illustrations, icons, and scalable graphics
- **WebP**: Modern format for photos (use with JPEG fallback)
- **PNG**: For images requiring transparency
- **JPEG**: For photographs without transparency

## Optimization

All images should be optimized before adding:
- Compress images (TinyPNG, ImageOptim, etc.)
- Use appropriate dimensions (don't upload oversized images)
- Consider responsive images with srcset

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `ingredient-network-visualization.svg`
- `food-comparison-italy-usa.png`
- `author-profile.jpg`

## Accessibility

Always provide descriptive alt text in HTML for all images.
